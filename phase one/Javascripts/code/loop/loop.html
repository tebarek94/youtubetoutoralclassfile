What Are Loops?

In JavaScript, loops let you repeat actions without writing the same code again and again. They help you run a block of code multiple times automatically.

Types of Loops:

  • for loop – runs a block of code a specific number of times
  • while loop – runs as long as a condition is true
  • do...while loop – runs at least once before checking the condition
